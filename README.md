# Symbolic-TDA Time-Series Analysis

This project develops a scientific-computing pipeline for symbolic-topological time-series analysis.

The planed pipeline is:

1. Load financial time-series data
2. Compute log returns
3. Normalize returns
4. Apply SAX sumbolic encoding
5. Construct transision graphs
6. Build weighed transition graphs
7. Construct filtrations and simplicial complexes
8. Compute persistent homology
9. Visualize results with ROOT
